HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Pretty much html is all about structuring data on a webpage, both visible and invisible
  • functions are separated by the greater-than less-than symbols and are closed with the same function handle

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The format for adding elements to the scrip is: div{property:value, property: value, ...}
  • Lots of different ways to add colors like 7 number codes, or just "blue"

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • to commit changes to github, follow these steps:
  • git add -A
  • git commit -m “updated project README file”
  • git pull origin main
  • git push origin feature/project-README

JavaScript

  • script runs from top down
  • define variables with : var ___ = ____;
  • can post things to the google chrome deve log by writing console.log(___)
